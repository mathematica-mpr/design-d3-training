PRIVATE HEALTH INSURANCE IN US STATES FROM 2016-2020

The following data visualization aims to compare the percentage of each state population with private health insurance with the average percentage of the country's population with private health insurance.

HOW TO USE THIS VISUALIZATION

STATE SELECTION: Use dropdown menu to select states for display in the bar chart. It is recommended to limit selection to 8 states for maximum visibility.

YEAR SELECTION: Use dropdown menu to select year to retrieve state data from. This can be used to observe changes in deviation over time.

VIEW SPECIFIC PERCENTAGE VALUES: Hover over the bar to view additional details about the data
