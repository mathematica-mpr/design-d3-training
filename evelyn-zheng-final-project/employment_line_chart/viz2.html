EMPLOYMENT PERCENTAGES IN US STATES FROM 2016-2020

The following data visualization explores changes in employment within US state populations from 2016-2020. It calculates the average employment percentage of counties in each state for each year.

HOW AVERAGE IS CALCULATED

Average for {state} = Î£(Percent of {county} Employed) / {Number of counties}

HOW TO USE THIS VISUALIZATION

STATE SELECTION: Use dropdown menu to select states for display in the line chart. It is recommended to limit selection to 3 states for maximum visibility.

COUNTRY LINE: Use "country" checkbox in the state dropdown menu to display the line graph for the average employment percentage of the US population. This can be used to compare state employment changes to the national changes.

VIEW SPECIFIC PERCENTAGE VALUES: Hover over the data points on the lines to view additional details about the data
